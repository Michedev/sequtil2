## sequtil2
nim package that provide additional functions for sequences

#### Function provided by sequtils2:
- Cartesian product (**)
- count
- filterWithIndex
- first
- firstWithIndex
- forEach
- forEachIndexed
- last
- lastWithIndex
- mapWithIndex
- maxBy
- minBy
- product
- reduce
- reduceReverse
- reverse
- sortBy
- sum
- uniques
- zipWithIndex



Documentation [link](http://htmlpreview.github.io/?https://github.com/Michedev/sequtils2/blob/master/sequtils2.html)
